一、注册问题#

注册显示我的邮箱已经注册过#

  • 可能是确实注册了，但是你没收到邮件
  • 还有很大概率是因为你的IP地址不干净，具体详情请看->解决一切注册账号IP限制问题

注册显示成功但是没有收到邮件#

  • 检查垃圾箱
  • 因为注册人数过多他的邮件可能延时发送

二、使用问题#

ssh无法连接服务器#

  • 出现ssh: connect to host s5.serv00.com port 22: Connection timed out，意味着连接超时，可以尝试切换节点，或者关闭代理来连接，实在不行的话使用webssh来连接

服务器经常自动重启导致vless连接失败怎么办#

  • ssh进入服务器，执行pm2 save,如果还不行的话删除域名之下vless以及serv00-vless文件夹，切换端口并重新部署
  • 实在不行重新注册个serv00账号，门槛极低，临时邮箱即可注册

节点速度太慢，还容易被封怎么办#

  • 可以尝试套一下cloudflare的CDN，不过这样的话也会让他的IP变得不干净，还不如直接使用cloudflare优选域名来搭建优选订阅器
  • 他的优点是IP相对与很多机场和cloudflare的优选IP来说会干净很多
  • 学会取长补短，比如平时情况使用机场，打算注册某些账号的时候切换serv00

如何获取root权限#

  • 获取不了，serv00就是个免费的虚拟机，没有办法获取root权限
  • 如果有朋友有类似的需求，可以尝试注册甲骨文和谷歌云的VPS，他们都有免费活动

我没有自定义域名怎么办#

  • 官方提供了一个<username>.serv00.com的域名供你使用，只不过无法自定义DNS，如果需求不大，也可以直接使用官方的，或者注册一个ClouDNS的免费域名

服务器如何重制#

  • 无法直接重制，可以删除你部署的项目，对应的端口，以及绑定的域名
  • 或者直接重新注册一个，临时邮箱据可以注册，门槛很低

如何搭建某某面板#

  • 可以自行查找一些教程，因为没有root权限，安装很多依赖，有些还伴随着需要修改代码，这块不详细解释

如何切换php版本号#

  • 进入domains–域名文件下，添加一个文档.htaccess 点击:heavy_plus_sign:New empty file 右击文档 .htaccess，依次点击：View/Edit -Choose other–Text Editor 然后将代码放入文件中：AddType application/x-httpd-php56 .php 56为版本号，自行修改就可以
